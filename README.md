#  AI Resume Builder

An AI-powered Resume Builder application that helps users create, manage, and enhance professional resumes using Google Gemini AI. Users can build resumes through an intuitive interface, improve resume content with AI assistance, and export resumes in a professional format.

##  Features

###  Authentication & Security
- User Registration & Login
- JWT-based Authentication
- Secure Password Hashing using bcrypt
- Protected Routes and User Sessions

### Resume Management
- Create Multiple Resumes
- Edit and Update Resume Sections
- Personal Information Management
- Education Details
- Work Experience
- Projects
- Skills
- Certifications
- Professional Summary

###  AI-Powered Resume Enhancement
Integrated **Google Gemini API** to:

- Generate Professional Summaries
- Enhance Job Descriptions
- Improve Project Descriptions
- Optimize Work Experience Content
- Rewrite Resume Content with ATS-Friendly Language
- Suggest Better Keywords and Action Verbs

###  Resume Templates
- Multiple Resume Templates
- Live Preview
- Responsive Design
- Professional Layouts

###  Cloud Features
- Resume Data Stored in MongoDB
- User-specific Resume Management
- Secure Backend APIs
